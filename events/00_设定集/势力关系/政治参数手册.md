## 一、群众层（6 项）

群众参数描述**人的感受**——忠诚、愤怒、恐惧。按国家 × 种族 × 脉络层三维分组，约 100~130 个群体。

### 1. 忠诚度（0 ~ 100）

群体对当前统治秩序的服从意愿。不是"爱"，是"认"——承认这套秩序是目前最不坏的选择。

| 区间 | 含义 |
|---|---|
| 0~20 | 反抗期：不再承认合法性，准备推翻 |
| 21~40 | 离心期：服从是惯性而非信念 |
| 41~60 | 冷漠期：无感，不关心谁在台上 |
| 61~80 | 顺从期：服从因为"总得有人管" |
| 81~100 | 忠诚期：主动维护秩序 |

### 2. 好战度（-100 ~ 100）

群体支持对外开战（或反战）的强度。正值=主战，负值=求和。

| 区间 | 含义 |
|---|---|
| -100~-60 | 和平主义：宁可投降也不打 |
| -59~-20 | 厌战：战争代价太大，能避则避 |
| -19~20 | 观望：看情况 |
| 21~60 | 主战：有利可图 / 有仇必报 |
| 61~100 | 狂热：战争即净化 |

> 当劳资张力爆表时，工人的好战度会从"对外开战"转为"对资方开战"。

### 3. 怨气值（0 ~ 100）

群体压抑未释放的怨念总量。**单调递增**，只有事件释放（罢工、暴动、革命、制度改良）才能下降。

| 区间 | 含义 |
|---|---|
| 0~20 | 平静：没有系统性不满 |
| 21~40 | 低怨：偶有抱怨 |
| 41~60 | 中怨：开始影响日常决策 |
| 61~80 | 高怨：随时可能被点燃 |
| 81~100 | 临界：社会火药桶 |

> **核心机制**：只增不减，除非释放。每回合自然增量 = 0.5 × (100 - 忠诚度) / 100

### 4. 行动力（0 ~ 100）

群体被组织起来行动的效率。不是"想不想动"，是"能动多快"。

| 区间 | 含义 |
|---|---|
| 0~20 | 散沙：无法组织 |
| 21~40 | 松散：行动迟缓、容易瓦解 |
| 41~60 | 可动员：有行会 / 工会网络 |
| 61~80 | 高效：成熟组织体系 |
| 81~100 | 军事化：组织度接近军队 |

> 行动力 × 怨气值 = 实际威胁。

### 5. 话语权（0 ~ 100）

群体在政治话语中的权重。谁的声音被听到，谁的声音被忽略。

| 区间 | 含义 |
|---|---|
| 0~20 | 无声：政治体系完全不回应 |
| 21~40 | 低声：偶尔被关注 |
| 41~60 | 回响：声音能到达决策层 |
| 61~80 | 响亮：足以改变政策方向 |
| 81~100 | 震耳：声音本身就是权力 |

> 话语权充当公司层聚合权重——高话语权的群体在劳资聚合中占据更大权重。

### 6. 激进度（-100 ~ 100）

群体在"温和↔极端"轴上的立场偏移。正值=保守极端，负值=激进极端，0=温和中立。

| 区间 | 含义 |
|---|---|
| -100~-60 | 革命派：现有秩序必须被推翻 |
| -59~-20 | 改革派：需要大幅改革 |
| -19~20 | 温和派：视情况而定 |
| 21~60 | 保守派：小修即可 |
| 61~100 | 顽固派：任何改变都是威胁 |

> 群众层激进度为单维度——复杂的二维路线坐标到党派层才出现。

## 二、公司层（4 项）

公司参数描述**组织的能力**——生产力、议价权、冲突。劳资双轨嵌入公司层公式。

### 1. 生产稳定度（0 ~ 100）

公司维持稳定产出的可靠性。劳方愿意干、能干、不闹罢工，生产稳定度就高。

- `0~30` 产能崩溃 / 停摆 / 订单违约
- `30~50` 间歇性停工 / 交付延迟
- `80~100` 满产稳定 / 扩产可行

### 2. 议价权（0 ~ 100）

公司在政策桌上能拍多大的板子。话语权高的资方 + 不可替代的产业地位 + 大产值 = 政策时该公司说了算。

- `<20` 被政策边缘化
- `>70` 游说成功 / 法案倾斜
- `>85` 可单独否决不利法案

### 3. 政治捆绑度（-100 ~ 100）

公司作为政治行为体的整体立场偏移。劳资双方的激进度在这里被强行缝合，加上产业链战争修正。

- `>60` 公司公开支持反建制运动 / 资助激进党派
- `<-60` 公司公开支持极端保守派 / 资助镇压
- `|值| >80` 公司可能直接资助政变 / 暗杀

### 4. 劳资冲突程度（0 ~ 100）

公司内部劳资关系破裂的可能性。是直接的事件触发器——值高即罢工。

- `>60` 罢工
- `>80` 暴力冲突 / 工厂占领
- `>95` 劳资全面决裂

## 三、党派层（4 项）

党派参数描述**派系作为政治组织的能力**。路线升级为二维光谱（经济轴 + 威权轴）。

### 1. 席位（0 ~ 100）

党派在议会 / 元老院等正式权力机构中占据的份额。议席=制度内权力。

- `>50` 单独执政
- `30~50` 联合执政可能性高
- `<15` 边缘小党 / 被排除决策

### 2. 票仓（0 ~ 100）

党派在制度外的群众动员能力。能不能上街、能不能堵议会、能不能罢工瘫痪经济。

- `>70` 可发动全国性街头运动
- `>50` 可发动地区性罢工 / 游行
- `<20` 街头无人响应

### 3. 路线（二维：经济轴 -100~100 + 威权轴 -100~100）

党派在二维政治光谱上的坐标。

- **经济轴**（-100=左 / +100=右）：左=劳方利益 / 再分配 / 计划经济，右=资方利益 / 自由市场 / 私有化
- **威权轴**（-100=自由 / +100=威权）：威权=秩序至上 / 国家管控，自由=个体优先 / 反垄断 / 地方自治

四象限示例：

| 象限 | 坐标示意 | 典型政策 |
|---|---|---|
| 左威权 | (-50, +50) | 国有化、灵脉公有化 |
| 右威权 | (+50, +50) | 戒严令、垄断保护法 |
| 左自由 | (-50, -50) | 灰市合法化、工会自治 |
| 右自由 | (+50, -50) | 减税、解除管制 |

### 4. 凝聚力（0 ~ 100）

党派内部路线一致性。低=党内派系林立，随时可能分裂；高=铁板一块，路线一致。

- `<30` 党派分裂 / 出走 / 独立组党
- `30~50` 党内路线斗争公开化
- `>80` 党内统一行动力强，但易陷入教条

## 四、王国层（5 项）

王国参数描述**国家作为主权实体的能力**——外交、秩序、军力、经济、政权。

### 1. 外交实力（0 ~ 100）

王国在国际谈判桌上的实际分量。分量重=别国不敢欺负你，分量轻=被反复施压。

- `<30` 被外交施压 / 被迫让步 / 割地
- `<50` 谈判中被动
- `>70` 主动施压他国 / 条约主导
- `>85` 区域霸权 / 制定国际规则

### 2. 社会稳定度（0 ~ 100）

王国境内的社会秩序水平。低=暴动 / 罢工 / 骚乱不断，高=维稳有效。

- `<30` 全国性暴动 / 革命危机
- `<50` 多地骚乱 / 街头对峙
- `>70` 维稳有效 / 异见被压制
- `>85` 社会高度顺从

### 3. 军力（0 ~ 100）

王国的实际作战能力。装备、动员、士气，三样齐了才叫军力。

- `<30` 被入侵 / 被迫投降 / 边境失守
- `<50` 防御吃力
- `>70` 主动开战 / 军事威慑有效
- `>85` 区域军事霸权

### 4. 经济稳定度（0 ~ 100）

王国经济的抗压能力。资源自主、产能稳定、内部不闹，经济就扛得住。

- `<30` 被制裁瘫痪 / 经济崩溃
- `<50` 被卡脖子 / 通胀失控
- `>70` 可承受制裁 / 经济自循环
- `>85` 可对他国施加经济制裁

### 5. 政治稳定度（0 ~ 100）

当权者的执政稳固程度。席位多、党内一条心、街头太平、对手党不极端，政权就稳。

- `<20` 政变 / 倒阁 / 政府倒台
- `<40` 执政危机 / 被迫妥协
- `>70` 政策推行顺畅
- `>85` 长期执政 / 可启动制度变革

## 计算方式

### 中间量与配置常量

#### M1. 怨气紧密度

```
怨气紧密度 = max(0, (劳方怨气值 - 40) / 60)   # 取值 0~1
```

来源：群众·怨气值（劳方）。怨气值低于 40 时不构成行动威胁。

#### M2. 劳资张力

```
劳资张力 = 0.25 × |劳方忠诚度 - 资方忠诚度|
        + 0.30 × |劳方怨气值 - 资方怨气值|
        + 0.20 × |劳方好战度 - 资方好战度|
        + 0.25 × |劳方激进度 - 资方激进度|
```

来源：全部四项均来自群众层（劳方/资方双轨各取一遍）。劳资两边在四个核心维度上的总落差。

#### M3. 公司路线坐标

```
公司经济倾向 = 200 × 0.7 × (资方话语权 / (劳方话语权 + 资方话语权) - 0.5)
            + 0.3 × (资方话语权 - 劳方话语权)
   分母为 0（劳资话语权均为 0）时，第一项约定为 0
   值域 [-100, 100]，左右对称（第一项 [-70,70] + 第二项 [-30,30]）

公司威权倾向 = clamp(资方好战度 - 劳资冲突程度, -100, 100)
```

来源：群众·话语权 / 好战度（劳资双轨）+ 公司·劳资冲突程度。被党派经济轴、威权轴、凝聚力公式引用。

#### K1. 军工链筛选规则

```
军工链公司 = 满足"产业链类型 = military"的全部公司
```

#### K2. 全国群众加权基准

```
群众参数全国加权 = Σ(参数值 × 群体人口) / Σ(群体人口)
```

#### K3. 全国公司加权基准

```
公司参数全国加权 = Σ(参数值 × 公司劳方人数) / Σ(公司劳方人数)
```

注：本版统一改用公司劳方人数加权（用于经济轴、威权轴、经济稳定度、社会稳定度等公司层聚合）。

### 群众层公式

#### 忠诚度

```
忠诚度每回合自然变化 = 0.1 × (经济周期参数 - 50)
                  + 0.2 × (消费比值 - 1) × 100

其中 消费比值 = 群体上回合消费 / 群体上上回合消费
```

变量来源：
- 经济周期参数 → 外生变量·经济模拟系统，默认 50
- 群体消费数据 → 外生变量·经济模拟系统（按群体逐回合记录）

#### 好战度

群众层基础参数，由事件直接修改，无内部计算公式。
驱动因素：外敌威胁↑ / 战争伤亡↓ / 军工利润↑(资方) / 反战宣传↓

#### 怨气值

```
怨气值每回合自然增量 = 0.5 × (100 - 忠诚度) / 100
```

释放方式：罢工(-20~-40)、暴动(-40~-60)、革命(-60~-80)、制度改良(-10~-30)

#### 行动力

群众层基础参数，由事件直接修改，无内部计算公式。
驱动因素：行会 / 工会存在↑ / 组织被镇压↓ / 共同敌人↑

#### 话语权

群众层基础参数，由事件直接修改，无内部计算公式。
驱动因素：财富↑ / 人口少↓ / 垄断地位↑ / 被边缘化↓

#### 激进度

群众层基础参数，由事件直接修改，无内部计算公式。
正值=保守极端（既得利益），负值=激进极端（被压迫）

### 公司层公式

#### 生产稳定度

```
生产稳定度 = 0.40 × 劳方忠诚度
         + 0.30 × (100 - 劳方怨气值)
         + 0.20 × (100 - 劳方行动力 × 怨气紧密度)
         + 0.10 × (100 - 劳资张力)
```

变量来源：
- 劳方忠诚度 / 劳方怨气值 / 劳方行动力 → 群众层（劳方双轨）
- 怨气紧密度 → 中间量 M1
- 劳资张力 → 中间量 M2

#### 议价权

```
议价权 = Σ(资方话语权_该种族 × 公司内资方该种族雇员数)
     / Σ(公司内资方雇员数)
```

即：按公司内部资方雇员的种族构成，对各种族资方话语权做加权平均。

变量来源：
- 资方话语权（按种族细分）→ 群众层参数五（资方双轨 × 种族维度）
- 公司内资方各种族雇员数 → 公司基础数据（外生）

#### 政治捆绑度

```
政治捆绑度 = 0.50 × 劳方激进度
         + 0.50 × 资方激进度
         + 产业链战争修正 × 资方好战度
```

变量来源：
- 劳方 / 资方激进度 → 群众层参数六
- 资方好战度 → 群众层参数二
- 产业链战争修正 → 配置常量·YAML：military +0.15 / alchemy -0.10 / magitech +0.05 等

#### 劳资冲突程度

```
劳资冲突程度 = 0.35 × 劳方怨气值
          + 0.20 × 劳方行动力
          + 0.15 × (100 - 劳方忠诚度)
          + 0.30 × 劳资张力
```

变量来源：
- 劳方怨气值 / 行动力 / 忠诚度 → 群众层（劳方双轨）
- 劳资张力 → 中间量 M2

### 党派层公式

#### 席位

```
席位 = Σ(产业链产值系数_i × 公司议价权_i × 公司劳方人数_i)
     / Σ(公司劳方人数_i)

其中 i 遍历该党派旗下所有公司
产业链产值系数按公司逐个乘入，跨产业链的党派天然得到混合估值
```

变量来源：
- 公司议价权 → 公司层参数二
- 产业链产值系数 → 配置常量·YAML
- 公司劳方人数 → 公司基础数据（外生）

#### 票仓

```
原始票仓 = (5/9) × W(劳方行动力)
        + (5/9) × W(|公司政治捆绑度|)
        - (1/9) × W(劳方怨气值)

其中 W(x) = Σ(x_i × 公司劳方人数_i) / Σ(公司劳方人数_i)
i 遍历该党派旗下所有公司

票仓 = clamp(原始票仓, 0, 100)
```

变量来源：
- 劳方行动力 / 劳方怨气值 → 群众层（劳方双轨）
- 公司政治捆绑度 → 公司层参数三（取绝对值，反建制立场不分左右）
- 系数说明：从用户原述 0.5 / 0.5 / -0.1 重缩放为 5/9、5/9、-1/9，使 |系数和| = 1

#### 路线（二维）

```
【公司经济倾向】（每家公司一个值，定义见 M3）
经济倾向 = 200 × 0.7 × (资方话语权 / (劳方话语权 + 资方话语权) - 0.5)
        + 0.3 × (资方话语权 - 劳方话语权)
   分母为 0 时第一项约定为 0
   值域 [-100, 100]，左右对称

【公司威权倾向】（每家公司一个值，定义见 M3）
威权倾向 = clamp(资方好战度 - 劳资冲突程度, -100, 100)

【经济轴】（党派路线第一维度）
经济轴 = Σ(公司经济倾向_i × 公司劳方人数_i) / Σ(公司劳方人数_i)

【威权轴】（党派路线第二维度）
威权轴 = Σ(公司威权倾向_i × 公司劳方人数_i) / Σ(公司劳方人数_i)

i 遍历该党派旗下所有公司，加权基准为劳方人数
```

变量来源：
- 资方 / 劳方话语权 → 群众层参数五
- 资方好战度 → 群众层参数二
- 公司劳资冲突程度 → 公司层参数四
- 公司经济倾向 / 威权倾向 → 中间量 M3（公司路线坐标）

#### 凝聚力

```
路线方差 = sqrt(Var(公司经济倾向) + Var(公司威权倾向))    # 按公司劳方人数加权
凝聚力 = 100 - clamp(路线方差, 0, 100)
```

变量来源：
- 公司经济倾向 / 公司威权倾向 → 中间量 M3（公司路线坐标）
- 加权基准 → 公司劳方人数

### 王国层公式

#### 外交实力

```
外交实力 = 0.35 × 军力
        + 0.65 × 执政党席位
```

变量来源：
- 军力 → 王国层自身参数（内部互引）
- 执政党席位 → 党派层参数一

#### 社会稳定度

```
社会稳定度 = 0.35 × 执政党席位
          + 0.25 × (100 - max(在野党票仓))
          + 0.25 × (100 - 全国公司加权平均(劳资冲突程度))
          + 0.15 × (100 - 全国群众加权平均怨气值)
```

变量来源：
- 执政党席位 / 在野党票仓 → 党派层参数一、二
- 全国公司加权平均(劳资冲突程度) → 公司层参数四 + K3 加权基准（劳方人数）
- 全国群众加权平均怨气值 → 群众层参数三 + K2 加权基准（群体人口）

#### 军力

```
军力 = 0.35 × 军工链公司加权(生产稳定度, 权重=公司劳方人数)
     + 0.30 × 执政党票仓 × (执政党旗下劳方人数 / 全国劳方人数)
     + 0.35 × 全国群众加权(好战度, 权重=种族人数)
```

注：好战度按字面写，不做 +50/+100 偏移；其取值 -100~100 直接进入聚合。

变量来源：
- 军工链公司 → 中间量 K1 筛选规则；公司生产稳定度 → 公司层参数一
- 执政党票仓 → 党派层参数二；劳方人数 → 公司基础数据
- 全国群众加权(好战度) → 群众层参数二 + K2 加权基准（按种族人数）

#### 经济稳定度

```
经济稳定度 = 0.65 × 全国公司加权(生产稳定度, 权重=公司劳方人数)
          + 0.35 × (100 - 全国公司加权(劳资冲突程度, 权重=公司劳方人数))
```

变量来源：
- 全国公司加权(生产稳定度 / 劳资冲突程度) → 公司层参数一、四 + K3 加权基准

#### 政治稳定度

```
政治稳定度 = 0.35 × 执政党席位
          + 0.20 × 执政党凝聚力
          + 0.25 × 社会稳定度
          + 0.20 × (100 - clamp(执政党与最大反对党的党派对立度 / 2, 0, 100))

党派对立度 = sqrt((A_经济轴 - B_经济轴)² + (A_威权轴 - B_威权轴)²)
```

变量来源：
- 执政党席位 / 凝聚力 → 党派层参数一、四
- 社会稳定度 → 王国层参数二
- 党派对立度 → 党派层衍生值
