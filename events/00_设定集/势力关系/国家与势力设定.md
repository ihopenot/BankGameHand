# 国家与势力：辉脉大陆十五国

> **BankGame 宇宙基础文件·地缘篇** — 每个种族对应一个主权实体，但国境之内种族杂处。原料产地决定贸易格局，贸易格局决定政治张力。
>
> 设计哲学：**资源诅咒 × 地缘博弈** — 谁控制了原力矿晶，谁就扼住了全大陆的喉咙。但垄断矿脉的矮人，却喝不到一口灵泉水。

---

## 一、大陆总览

**辉脉大陆（Luminous Vein Continent）**，总面积约等于现实世界的亚欧大陆。大陆中央纵贯一条**大灵脉**（Grand Leyline），从北境火山群直抵南方冥渊，沿途分支如血管般铺展。

大灵脉的魔力丰度分布不均，形成三个同心地带：
- **内环**（大灵脉主脉两侧200里）：魔力最浓，原料产出最丰，也是大国争霸的核心战场
- **中环**（主脉200-600里）：魔力中等，适合灵谷种植和灵蛛饲养，是产粮区
- **外环**（主脉600里以外）：魔力稀薄，但各有特产——沙漠的蜥蜴人、荒原的死灵、深海的晶力

**总人口：10,000**，分布在15个主权实体中。

## 三、各国详述

### 3.1 自由城邦联盟（Free Cities League）

> *"这座城不生产魔力——它流转魔力。每一条辉币都经过我们的手，每一次交易都在我们的账本上。"*
> —— 维克多·麦迪逊，第七金库掌钥人

| 维度 | 设定 |
|------|------|
| **主导种族** | 人类（48%，1,536人） |
| **其他种族** | 哥布林 18%、侏儒 10%、狗头人 8%、兽族 5%、蜥蜴人 4%、其他 7% |
| **领土** | 大陆中央平原，大灵脉中段最宽阔处 |
| **首都** | 辉币城（Lumenheim），建在大灵脉最粗的节点之上 |
| **原料产出** | 灵谷（平原灌脉农业，全大陆产量第二） |
| **政体** | 五大金库掌钥人组成的"钥匙会议"，形式上选举产生，实际由金库资本决定 |
| **核心矛盾** | 商人寡头 vs 边垦农民；金库利润 vs 社会稳定 |

**种族构成逻辑**：人类是金融家和管理者，哥布林负责城市运转（回收、搬运、小贩），侏儒经营工坊，狗头人充当地下运维员。兽族是码头工人和搬运工。蜥蜴人在城内经营量化基金。

**原料产出细节**：联盟平原是大灵脉最宽广的漫射区，魔力均匀渗透土壤，形成全大陆最肥沃的灌脉农田。灵谷产量占全大陆35%。

### 3.2 灰爪回收联邦（Grey Claw Recycling Federation）

> *"你们叫它垃圾。我叫它库存。"*
> —— 烬牙，灰市回收行会行首

| 维度 | 设定 |
|------|------|
| **主导种族** | 哥布林（62%，682人） |
| **其他种族** | 人类 12%、死灵 10%、兽族 6%、狗头人 5%、其他 5% |
| **领土** | 不是一个连续领土——各城市的地下层、废弃矿道、荒原边缘的聚落，由灰色物流线连接 |
| **首都** | 烬墟城（Ashmound），建在一座被灵脉遗弃的旧矿坑之上 |
| **原料产出** | 废弃魔导残片（全大陆唯一来源——城市废弃物的回收提纯） |
| **政体** | 三大行会（回收行会、灰色物流行会、翻新行会）轮值执政 |
| **核心矛盾** | 正规经济 vs 灰色经济；回收权 vs 产权争议 |

**种族构成逻辑**：哥布林是回收业的灵魂。人类中的一部分是边垦人，在灰市讨生活。死灵是被正规经济排斥的底层。兽族从事重体力回收作业。

**原料产出细节**：废弃魔导残片不是"矿藏"——它是对全大陆消费残渣的二次提取。每个城市的废品都是联邦的"矿区"。哥布林的鼻子能嗅出魔力残留的痕迹，这是其他种族无法复制的天赋。

### 3.3 炉心王国（Hearth-Core Kingdom）

> *"我们的锤子敲开矿脉，我们的脊梁撑起天空。辉币？那是矿脉流出来的，不是金库印出来的。"*
> —— 铁砧·深炉，炉心王

| 维度 | 设定 |
|------|------|
| **主导种族** | 矮人（48%，696人） |
| **其他种族** | 兽族 15%、人类 12%、狗头人 10%、哥布林 6%、死灵 4%、其他 5% |
| **领土** | 北境群山，大灵脉北段深入地壳处，矿脉最密集 |
| **首都** | 炉心城（Hearthcore），建造在活火山内部的巨型熔炉都市 |
| **原料产出** | 原力矿晶（全大陆产量60%）、灵脉石（全大陆产量30%） |
| **政体** | 世袭炉心王 + 三大氏族（炉矮人、铸矮人、深矮人）议会 |
| **核心矛盾** | 矿工兄弟会 vs 炉心王廷；死灵替代潮 vs 矮人就业；矿脉枯竭恐慌 |

**种族构成逻辑**：矮人是矿主和主力矿工。兽族是搬运工和初级矿工。人类是贸易中间商。狗头人负责矿道基建维护。哥布林在矿渣堆里拾荒。死灵是矿场主引入的廉价替代劳动力——这是矮人内部最大的撕裂。

**原料产出细节**：大灵脉在北境深入地壳，魔力结晶化程度最高，形成密集的原力矿脉。炉心王国的矿井延伸至地下三千米，最深层已接近灵脉核心。灵脉石是矿脉周围魔力自然凝结的建筑石材，筑城砖的上游原料。

**地缘意义**：原力矿晶是全产业链的起点。炉心王国一旦减产，精炼魔力断供，全大陆魔导器停产。这是"谁控制了矿脉，谁就扼住了大陆的喉咙"的具象化。

> *"灵蛛是我们的姐妹，灵谷是我们的血脉。城里的长老们在乎辉币，我们只在乎——明天灵蛛还能吐丝吗？"*
> —— 金鬃·狮心，大酋长

| 维度 | 设定 |
|------|------|
| **主导种族** | 兽族（52%，650人） |
| **其他种族** | 人类 15%、哥布林 10%、萨特羊人 8%、侏儒 5%、其他 10% |
| **领土** | 东部大草原，大灵脉东段分支漫射区，温暖湿润 |
| **首都** | 万灵帐（Spirit Lodge），每年随灵蛛迁徙路线移动的游牧王廷 |
| **原料产出** | 灵蛛丝（全大陆唯一产地）、灵谷（全大陆产量30%） |
| **政体** | 酋长联席会议，三大兽族氏族（狮族/狼族/熊族）轮流主持 |
| **核心矛盾** | 游牧传统 vs 定居现代化；灵蛛栖息地 vs 灵谷扩张；狮族霸权 vs 狼/熊族平权 |

**种族构成逻辑**：兽族是灵蛛饲养世家和灵谷种植者。人类是定居贸易商。哥布林是边缘拾荒者。萨特羊人在草原上经营酒馆和体验经济。侏儒偶尔来收购灵蛛丝进行加工。

**原料产出细节**：灵蛛只在大草原的魔力漫射区吐丝——灵蛛的腺体需要微量的环境魔力才能产出附魔级丝线。这种生态依赖无法人工复制，使得灵蛛丝成为"地理垄断"型原料。灵谷是大草原的传统作物，兽族积累了千年灌脉种植经验。

### 3.5 螺牙共和国（Screw-Tooth Republic）

> *"我们不挖矿、不养蛛、不种田。我们做的事更值钱——我们把原力矿晶变成魔导核心，把灵蛛丝变成附魔纤维。原料是别人的，增值是我们的。"*
> —— 齿轮·螺牙，共和国首席工程师

| 维度 | 设定 |
|------|------|
| **主导种族** | 侏儒（52%，494人） |
| **其他种族** | 人类 15%、精灵 8%、狗头人 7%、兽族 5%、蜥蜴人 5%、其他 8% |
| **领土** | 西部丘陵工坊群，大灵脉西段分支，魔力中等但地形适合工坊建设 |
| **首都** | 螺牙城（Gearhaven），全大陆最大的阵纹研发中心 |
| **原料产出** | 无——完全的加工经济 |
| **政体** | 首席工程师 + 技术院士院（按阵纹专利贡献度投票） |
| **核心矛盾** | 专利开放 vs 知识垄断；创新速度 vs 精灵专利壁垒；工程侏儒 vs 炼金侏儒的路线之争 |

**种族构成逻辑**：侏儒是阵纹工程师和炼金师。人类是项目经理和销售员。精灵是专利许可方（坐收专利费）。狗头人负责设备运维。兽族从事重体力工坊操作。蜥蜴人经营数据服务。

**经济模式**：螺牙共和国是全大陆的"加工车间"。所有原料都需要运到这里加工——魔导核心的蚀刻、附魔纤维的编织、灵馐剂的调配。侏儒不控制原料，但他们控制**转化能力**。切断螺牙的工坊，矿晶就只是矿晶，永远变不成核心。

### 3.6 银叶诸领（Silver Leaf Dominions）

> *"这些法袍已经穿了三百年。你们的公司呢？"*
> —— 银叶·晨星，诸领大长老

| 维度 | 设定 |
|------|------|
| **主导种族** | 精灵（56%，420人） |
| **其他种族** | 侏儒 12%、人类 10%、龙裔 5%、提夫林 4%、其他 13% |
| **领土** | 西北古森林，大灵脉西北分支，月华浓度最高 |
| **首都** | 银叶城（Silverleaf），千年古树中空形成的树城 |
| **原料产出** | 月华绒（全大陆唯一产地——银月蛾仅栖息于古森林的魔力月华区） |
| **政体** | 三大家族（高等精灵/林精灵/暗精灵）长老联席，高等精灵事实主导 |
| **核心矛盾** | 高等精灵专利贵族 vs 林精灵传统派 vs 暗精灵禁忌研究者；旧钱稳定 vs 创新风险 |

**种族构成逻辑**：精灵是贵族和专利持有者。侏儒是被雇佣的工程师。人类是精灵庄园的管理者。龙裔是精灵的"储备锚"——龙裔的晶力与精灵的专利是辉币信用的双重支撑。提夫林在精灵的影子中经营灰色金融。

**原料产出细节**：月华绒的生产依赖银月蛾，而银月蛾只在古森林中魔力月华浓度极高的区域吐丝。这个生态条件无法人工复制，精灵对月华绒的垄断比兽族对灵蛛丝更绝对——定制法袍的上游原料100%掌握在精灵手中。

### 3.7 冥渊自治区（Abyssal Autonomous Zone）

> *"我们不需要光。我们不需要热。我们只需要——不被遗忘。"*
> —— 缚灵者·七号，冥渊执委会发言人

| 维度 | 设定 |
|------|------|
| **主导种族** | 死灵（45%，306人） |
| **其他种族** | 人类 15%、矮人 12%、哥布林 10%、狗头人 8%、兽族 5%、其他 5% |
| **领土** | 南方冥渊荒原，大灵脉南段终点——灵脉在此深潜入地幔，地表魔力被抽干形成死域 |
| **首都** | 冥渊城（The Abyss），建造在旧矿场废墟上的钢铁堡垒 |
| **原料产出** | 冥铁（全大陆唯一产地——死域地壳中魔力真空区自然凝结的阴寒铁矿） |
| **政体** | 执行委员会（冥匠主导，缚灵者辅政，回魂尸无参政权） |
| **核心矛盾** | 冥匠精英 vs 回魂尸底层；死灵劳动力 vs 活体劳动力；自治权 vs 元老院管辖 |

**种族构成逻辑**：死灵是主体，但内部三等分——冥匠（有意识的极少数）是统治者，缚灵者（半意识体）是中层执行者，回魂尸（无意识体）是底层劳动力。矮人是来挖冥铁矿的矿工。人类是冥渊的"贸易代理人"。哥布林回收死灵的废弃物资。

**原料产出细节**：冥铁只产自死域——灵脉被抽干后，魔力真空导致地壳中阴寒铁矿自然凝结。这是一种"诅咒矿物"：只有在魔力匮乏的环境中才能形成。冥铁是军工链的唯一矿石来源。

### 3.8 赤沙商盟（Red Sand Merchant League）

| **领土** | 南部沙漠绿洲带，大灵脉的魔力在此蒸发殆尽，但地下的旧灵脉通道形成了天然贸易路线 |
| **首都** | 赤沙城（Redsand），全大陆最大的商品交易所所在地 |
| **原料产出** | 无——纯粹的贸易/金融/服务经济 |

**经济模式**：赤沙商盟是全大陆的"华尔街"和"迪拜"——不产一滴油，但控制着所有石油的定价权。商品交易所、期货市场、魔力衍生品交易所全部设在这里。蜥蜴人的量化模型决定着全大陆的魔力期货价格。
