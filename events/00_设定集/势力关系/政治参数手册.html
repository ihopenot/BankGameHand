一、群众层（6项）

群众参数描述人的感受——忠诚、愤怒、恐惧。按国家×种族×脉络层三维分组，约100~130个群体。

忠诚度 0 ~ 100
群体对当前统治秩序的服从意愿。不是"爱"，是"认"——承认这套秩序是目前最不坏的选择。
  • 0~20 反抗期 — 不再承认合法性，准备推翻
  • 21~40 离心期 — 服从是惯性而非信念
  • 41~60 冷漠期 — 无感，不关心谁在台上
  • 61~80 顺从期 — 服从因为"总得有人管"
  • 81~100 忠诚期 — 主动维护秩序
计算方式 ↓
好战度 -100 ~ 100
群体支持对外开战（或反战）的强度。正值=主战，负值=求和。
  • -100~-60 和平主义 — 宁可投降也不打
  • -59~-20 厌战 — 战争代价太大，能避则避
  • -19~20 观望 — 看情况
  • 21~60 主战 — 有利可图/有仇必报
  • 61~100 狂热 — 战争即净化

当劳资张力爆表时，工人的好战度会从"对外开战"转为"对资方开战"。

计算方式 ↓
怨气值 0 ~ 100
群体压抑未释放的怨念总量。单调递增，只有事件释放（罢工、暴动、革命、制度改良）才能下降。
  • 0~20 平静 — 没有系统性不满
  • 21~40 低怨 — 偶有抱怨
  • 41~60 中怨 — 开始影响日常决策
  • 61~80 高怨 — 随时可能被点燃
  • 81~100 临界 — 社会火药桶

核心机制：只增不减，除非释放。每回合自然增量 = 0.5 × (100 - 忠诚度) / 100 × 生存压力系数

计算方式 ↓
行动力 0 ~ 100
群体被组织起来行动的效率。不是"想不想动"，是"能动多快"。
  • 0~20 散沙 — 无法组织
  • 21~40 松散 — 行动迟缓、容易瓦解
  • 41~60 可动员 — 有行会/工会网络
  • 61~80 高效 — 成熟组织体系
  • 81~100 军事化 — 组织度接近军队

行动力 × 怨气值 = 实际威胁。

计算方式 ↓
话语权 0 ~ 100
群体在政治话语中的权重。谁的声音被听到，谁的声音被忽略。
  • 0~20 无声 — 政治体系完全不回应
  • 21~40 低声 — 偶尔被关注
  • 41~60 回响 — 声音能到达决策层
  • 61~80 响亮 — 足以改变政策方向
  • 81~100 震耳 — 声音本身就是权力

话语权充当公司层聚合权重——高话语权的群体在劳资聚合中占据更大权重。

计算方式 ↓
激进度 -100 ~ 100
群体在"温和↔极端"轴上的立场偏移。正值=保守极端，负值=激进极端，0=温和中立。
  • -100~-60 革命派 — 现有秩序必须被推翻
  • -59~-20 改革派 — 需要大幅改革
  • -19~20 温和派 — 视情况而定
  • 21~60 保守派 — 小修即可
  • 61~100 顽固派 — 任何改变都是威胁

群众层激进度为单维度——复杂的二维路线坐标到党派层才出现。

二、公司层（4项）

公司参数描述组织的能力——生产力、议价权、冲突。劳资双轨嵌入公司层公式。

生产稳定度 0 ~ 100
公司维持稳定产出的可靠性。劳方愿意干、能干、不闹罢工，生产稳定度就高。
  • 0~30 产能崩溃/停摆/订单违约
  • 30~50 间歇性停工/交付延迟
  • 80~100 满产稳定/扩产可行
计算方式 ↓
议价权 0 ~ 100
公司在政策桌上能拍多大的板子。话语权高的资方+不可替代的产业地位+大产值=政策时该公司说了算。
  • <20 被政策边缘化
  • >70 游说成功/法案倾斜
  • >85 可单独否决不利法案
计算方式 ↓
政治捆绑度 -100 ~ 100
公司作为政治行为体的整体立场偏移。劳资双方的激进度在这里被强行缝合，加上产业链战争修正。
  • >60 公司公开支持反建制运动/资助激进党派
  • <-60 公司公开支持极端保守派/资助镇压
  • |值| >80 公司可能直接资助政变/暗杀
计算方式 ↓
劳资冲突程度 0 ~ 100
公司内部劳资关系破裂的可能性。是直接的事件触发器——值高即罢工。
  • >60 罢工
  • >80 暴力冲突/工厂占领
  • >95 劳资全面决裂

三、党派层（4项）

党派参数描述派系作为政治组织的能力。路线升级为二维光谱（经济轴+威权轴）。

席位 0 ~ 100
党派在议会/元老院等正式权力机构中占据的份额。议席=制度内权力。
  • >50 单独执政
  • 30~50 联合执政可能性高
  • <15 边缘小党/被排除决策
计算方式 ↓
票仓 0 ~ 100
党派在制度外的群众动员能力。能不能上街、能不能堵议会、能不能罢工瘫痪经济。
  • >70 可发动全国性街头运动
  • >50 可发动地区性罢工/游行
  • <20 街头无人响应
计算方式 ↓
路线 二维：经济轴(-100~100) + 威权轴(-100~100)
党派在二维政治光谱上的坐标。

经济轴（-100=左/+100=右）：左=劳方利益/再分配/计划经济，右=资方利益/自由市场/私有化

威权轴（-100=自由/+100=威权）：威权=秩序至上/国家管控，自由=个体优先/反垄断/地方自治

  • 左威权（-50,+50）：国有化、灵脉公有化
  • 右威权（+50,+50）：戒严令、垄断保护法
  • 左自由（-50,-50）：灰市合法化、工会自治
  • 右自由（+50,-50）：减税、解除管制
计算方式 ↓
凝聚力 0 ~ 100
党派内部路线一致性。低=党内派系林立，随时可能分裂；高=铁板一块，路线一致。
  • <30 党派分裂/出走/独立组党
  • 30~50 党内路线斗争公开化
  • >80 党内统一行动力强，但易陷入教条

四、王国层（5项）

王国参数描述国家作为主权实体的能力——外交、秩序、军力、经济、政权。

外交实力 0 ~ 100
王国在国际谈判桌上的实际分量。分量重=别国不敢欺负你，分量轻=被反复施压。
  • <30 被外交施压/被迫让步/割地
  • <50 谈判中被动
  • >70 主动施压他国/条约主导
  • >85 区域霸权/制定国际规则
计算方式 ↓
社会稳定度 0 ~ 100
王国境内的社会秩序水平。低=暴动/罢工/骚乱不断，高=维稳有效。
  • <30 全国性暴动/革命危机
  • <50 多地骚乱/街头对峙
  • >70 维稳有效/异见被压制
  • >85 社会高度顺从
计算方式 ↓
军力 0 ~ 100
王国的实际作战能力。装备、动员、士气，三样齐了才叫军力。
  • <30 被入侵/被迫投降/边境失守
  • <50 防御吃力
  • >70 主动开战/军事威慑有效
  • >85 区域军事霸权
计算方式 ↓
经济自给度 0 ~ 100
王国经济的抗压能力。资源自主、产能稳定、内部不闹，经济就扛得住。
  • <30 被制裁瘫痪/经济崩溃
  • <50 被卡脖子/通胀失控
  • >70 可承受制裁/经济自循环
  • >85 可对他国施加经济制裁
计算方式 ↓
政治稳定度 0 ~ 100
当权者的执政稳固程度。席位多、党内一条心、街头太平、对手党不极端，政权就稳。
  • <20 政变/倒阁/政府倒台
  • <40 执政危机/被迫妥协
  • >70 政策推行顺畅
  • >85 长期执政/可启动制度变革

计算方式

所有上层参数均由下层参数经公式计算得出，不是同名聚合。

中间量与配置常量

M1. 怨气紧密度

怨气紧密度 = max(0, (劳方怨气值 - 40) / 60)   # 取值 0~1
来源：群众·怨气值（劳方）。怨气值低于40时不构成行动威胁。

M2. 劳资张力

劳资张力 = 0.25 × |劳方忠诚度 - 资方忠诚度|
        + 0.30 × |劳方怨气值 - 资方怨气值|
        + 0.20 × |劳方好战度 - 资方好战度|
        + 0.25 × |劳方激进度 - 资方激进度|
来源：全部四项均来自群众层（劳方/资方双轨各取一遍）。劳资两边在四个核心维度上的总落差。

M3. 产值规模归一化

产值规模归一化 = clamp((公司产值 / 该国总产值) × 100, 0, 100)
来源：公司产值、该国总产值（外生·来自经济模拟系统）。

M4. 公司产值权重

公司产值权重 = 公司雇员数 × 产业链产值系数
来源：公司雇员数（外生）+ 产业链产值系数（配置常量·YAML）

M5. 公司政治捆绑度方向修正

设 R = 旗下公司政治捆绑度均值（按 M4 加权）
若 R > 0：经济轴 +10、威权轴 +10
若 R < 0：经济轴 -15、威权轴 -5
若 R = 0：均不变
来源：公司·政治捆绑度。被党派经济轴、威权轴公式引用。

M6. 公司路线坐标

公司经济倾向 = 100 × (资方话语权 / (劳方话语权 + 资方话语权 + 1) - 0.5) × 2
            + 0.30 × (-(劳方话语权 - 资方话语权))
            + 单公司政治捆绑度方向修正
   公司政治捆绑度 > 0 → 经济倾向 +10
   公司政治捆绑度 < 0 → 经济倾向 -15
   clamp 到 -100~100

公司威权倾向 = 0.40 × 资方好战度
            + 0.30 × 产业链卡位系数
            - 0.30 × 公司劳资冲突程度
            + 单公司政治捆绑度方向修正（威权部分）
   公司政治捆绑度 > 0 → +10；< 0 → -5
   clamp 到 -100~100
来源：群众·话语权/好战度（劳资双轨）+ 公司·政治捆绑度/劳资冲突程度 + 配置常量

K1. 资源自给率

资源自给率 = 该国本期自产关键原料数量 / 该国本期使用关键原料总数   # 取值 0~1
来源：经济模拟系统输出（外生中间量）

K2. 军工链筛选规则

军工链公司 = 满足"产业链类型 = military"的全部公司

K3. 全国群众加权基准

群众参数全国加权 = Σ(参数值 × 群体人口) / Σ(群体人口)

K4. 全国公司加权基准

公司参数全国加权 = Σ(参数值 × 公司产值权重) / Σ(公司产值权重)

群众层参数

忠诚度 ↑ 返回定义

忠诚度每回合自然变化 = 0.1 × (economic_confidence - 50) - 0.2 × wage_pressure
  • economic_confidence（经济信心）→ 外生变量·经济模拟系统，默认50
  • wage_pressure（工资压力）→ 外生变量·经济模拟系统，默认30

好战度 ↑ 返回定义

群众层基础参数，由事件直接修改，无内部计算公式。
驱动因素：外敌威胁↑ / 战争伤亡↓ / 军工利润↑(资方) / 反战宣传↓

怨气值 ↑ 返回定义

怨气值每回合自然增量 = 0.5 × (100 - 忠诚度) / 100 × 生存压力系数
释放方式：罢工(-20~-40)、暴动(-40~-60)、革命(-60~-80)、制度改良(-10~-30)
  • 忠诚度 → 群众层参数一
  • 生存压力系数 → 外生变量·经济/灾害/战争系统，默认1.0

行动力 ↑ 返回定义

群众层基础参数，由事件直接修改，无内部计算公式。
驱动因素：行会/工会存在↑ / 组织被镇压↓ / 共同敌人↑

话语权 ↑ 返回定义

群众层基础参数，由事件直接修改，无内部计算公式。
驱动因素：财富↑ / 人口少↓ / 垄断地位↑ / 被边缘化↓

激进度 ↑ 返回定义

群众层基础参数，由事件直接修改，无内部计算公式。
正值=保守极端（既得利益），负值=激进极端（被压迫）

公司层参数

生产稳定度 ↑ 返回定义

生产稳定度 = 0.40 × 劳方忠诚度
           + 0.30 × (100 - 劳方怨气值)
           + 0.20 × (100 - 劳方行动力 × 怨气紧密度)
           + 0.10 × (100 - 劳资张力)
  • 劳方忠诚度 / 劳方怨气值 / 劳方行动力 → 群众层（劳方双轨）
  • 怨气紧密度 → 中间量 M1
  • 劳资张力 → 中间量 M2

议价权 ↑ 返回定义

议价权 = 0.45 × 资方话语权
       + 0.30 × 产业链卡位系数      # 上游垄断90 / 上游普通60 / 中游40 / 下游20
       + 0.25 × 产值规模归一化      # 该国GDP占比
  • 资方话语权 → 群众层参数五（资方双轨）
  • 产业链卡位系数 → 配置常量·YAML
  • 产值规模归一化 → 中间量 M3

政治捆绑度 ↑ 返回定义

政治捆绑度 = 0.50 × 劳方激进度
           + 0.50 × 资方激进度
           + 产业链战争修正 × 资方好战度
  • 劳方/资方激进度 → 群众层参数六
  • 资方好战度 → 群众层参数二
  • 产业链战争修正 → 配置常量·YAML：military +0.15 / alchemy -0.10 / magitech +0.05 等

劳资冲突程度 ↑ 返回定义

劳资冲突程度 = 0.35 × 劳方怨气值
            + 0.20 × 劳方行动力
            + 0.15 × (100 - 劳方忠诚度)
            + 0.30 × 劳资张力
  • 劳方怨气值 / 行动力 / 忠诚度 → 群众层（劳方双轨）
  • 劳资张力 → 中间量 M2

党派层参数

席位 ↑ 返回定义

席位 = 100 × Σ(旗下公司议价权 × 公司产值权重) / Σ(全国所有公司议价权 × 公司产值权重)
  • 公司议价权 → 公司层参数二
  • 公司产值权重 → 中间量 M4

票仓 ↑ 返回定义

票仓 = 0.50 × 旗下公司加权(劳方行动力 × 公司劳资冲突程度归一化)
     + 0.30 × 旗下公司加权(|公司政治捆绑度|)
     + 0.20 × 旗下公司加权(劳方怨气值)

加权基准 = 公司雇员数
  • 劳方行动力 / 劳方怨气值 → 群众层（劳方双轨）
  • 公司劳资冲突程度 / 公司政治捆绑度 → 公司层参数三、四

路线（二维） ↑ 返回定义

经济轴 = 100 × (旗下公司加权(资方话语权 / (劳方话语权 + 资方话语权 + 1)) - 0.5) × 2
      + 0.30 × (-旗下公司加权(劳方话语权 - 资方话语权))
      + 公司政治捆绑度方向修正·经济部分

威权轴 = 0.40 × 旗下公司加权(资方好战度)
      + 0.30 × 旗下公司加权(产业链卡位系数)
      - 0.30 × 旗下公司加权(劳资冲突程度)
      + 公司政治捆绑度方向修正·威权部分

加权基准 = 公司雇员数 × 产业链产值系数（M4）
两轴均 clamp 到 -100~100
  • 资方/劳方话语权 → 群众层参数五
  • 资方好战度 → 群众层参数二
  • 产业链卡位系数 → 配置常量
  • 公司劳资冲突程度 → 公司层参数四
  • 公司政治捆绑度方向修正 → 中间量 M5

凝聚力 ↑ 返回定义

路线方差 = sqrt(Var(公司经济倾向) + Var(公司威权倾向))    # 按公司雇员数加权
凝聚力 = 100 - clamp(路线方差, 0, 100)
  • 公司经济倾向 / 公司威权倾向 → 中间量 M6（公司路线坐标）
  • 加权基准 → 公司雇员数

王国层参数

外交实力 ↑ 返回定义

外交实力 = 0.35 × 军力
        + 0.30 × 资源自给率 × 100
        + 0.20 × 执政党席位
        + 0.15 × 经济自给度
  • 军力 / 经济自给度 → 王国层自身参数（内部互引）
  • 资源自给率 → 中间量 K1
  • 执政党席位 → 党派层参数一

社会稳定度 ↑ 返回定义

社会稳定度 = 0.35 × 执政党席位
           + 0.25 × (100 - max(在野党票仓))
           + 0.25 × (100 - 全国公司加权平均(劳资冲突程度))
           + 0.15 × (100 - 全国群众加权平均怨气值)
  • 执政党席位 / 在野党票仓 → 党派层参数一、二
  • 全国公司加权平均(劳资冲突程度) → 公司层参数四 + K4 加权基准
  • 全国群众加权平均怨气值 → 群众层参数三 + K3 加权基准

军力 ↑ 返回定义

军力 = 0.35 × 军工链公司加权平均生产稳定度
     + 0.30 × 全国党派加权平均票仓 × 0.7
     + 0.25 × clamp(全国群众加权平均好战度 + 50, 0, 100)
     + 0.10 × 资源自给率 × 100
  • 军工链公司 → K2 筛选规则；公司生产稳定度 → 公司层参数一；加权基准=K4
  • 全国党派加权平均票仓 → 党派层参数二，加权基准=党派席位
  • 全国群众加权平均好战度 → 群众层参数二 + K3 加权基准
  • 资源自给率 → 中间量 K1

经济自给度 ↑ 返回定义

经济自给度 = 0.40 × 资源自给率 × 100
          + 0.35 × 全国公司加权平均生产稳定度
          + 0.25 × (100 - 全国公司加权平均劳资冲突程度)
  • 资源自给率 → 中间量 K1
  • 全国公司加权平均(生产稳定度 / 劳资冲突程度) → 公司层参数一、四 + K4 加权基准

政治稳定度 ↑ 返回定义

政治稳定度 = 0.35 × 执政党席位
          + 0.20 × 执政党凝聚力
          + 0.25 × 社会稳定度
          + 0.20 × (100 - clamp(执政党与最大反对党的党派对立度 / 2, 0, 100))

党派对立度 = sqrt((A_经济轴 - B_经济轴)² + (A_威权轴 - B_威权轴)²)
  • 执政党席位 / 凝聚力 → 党派层参数一、四
  • 社会稳定度 → 王国层参数二
  • 党派对立度 → 党派层衍生值
