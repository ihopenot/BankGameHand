# 工厂

## 属性

| 属性 | 说明 |
|------|------|
| 输入商品种类 | 单一原料类型（如"硅"） |
| 满产需求量 | 满负荷运转需要的原料量 |
| 输出商品种类 | 单一产品类型（如"芯片"） |
| 基础生产量 | 原料充足时的产出量 |
| 造价 | 建造工厂需要的现金 |
| 维护成本 | 每回合固定支出 |
| 建造时间 | 建造需要的回合数（整数） |

## 生产公式

### 产出量（原料品质影响，科技无关）

```
实际产出量 = 基础生产量 * 原料充足率 * 良品率加成

原料充足率 = min(实际采购量 / 满产需求量, 1.0)
良品率加成 = 1 + (等效质量 / 同类原料最高质量) * 10%
```

原料品质高 → 良品率高 → 产量多，最高加成 10%。

等效质量（多供应商采购时）：
```
等效质量 = sum(各批次质量 * 各批次数量) / 总数量
```

### 产出品质（科技影响，原料无关）

```
产出商品品质 = 科技排名系数 * 该商品品质上限

科技排名系数 = 该公司科技值 / 同类商品所有公司中最高科技值
```

科技最高的公司产出最高品质的产品，其他公司按比例。

### 生产成本

```
本回合生产成本 = 原料采购成本 + 工厂维护成本 + 工资

单位生产成本 = 本回合生产成本 / 实际产出量
```

减产时单位成本上升（维护成本和工资是固定支出）。
