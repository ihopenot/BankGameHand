# EconomyIndex

经济指数

## 传导链

```
经济周期指数 ──→ 标准利率 ──→ 贷款成本
      │
      └──→ 消费信心 ──→ 居民消费需求 ──→ 公司收入 ──→ 生产总值
                                                  ──→ 产能利用率
                                                  ──→ 通胀率
```

## 经济周期指数（外生驱动）

整个经济的"天气"，预设曲线驱动，公司和玩家只能适应。

取值范围：-1.0 ~ +1.0（-1 深度衰退，0 中性，+1 极度繁荣）

采用双周期叠加，模拟现实中短周期（库存周期）和长周期（产能周期）的叠加效应，使玩家能感受到"有周期性"但无法精确预测拐点：

```
经济周期指数(t) = clamp( A1 * sin(2π * t / T1 + φ1)
                        + A2 * sin(2π * t / T2 + φ2)
                        + noise(t),
                        -1.0, +1.0 )

短周期：
  A1 = 主振幅（如 0.4~0.5），控制短期波动
  T1 = 短周期长度（如 20~30 回合）
  φ1 = 初始相位（随机）

长周期：
  A2 = 次振幅（如 0.2~0.3），控制长期趋势
  T2 = 长周期长度（如 50~80 回合），与 T1 不构成简单整数比
  φ2 = 初始相位（随机，独立于 φ1）

noise = 小幅随机扰动（如 ±0.05），让同周期位置每回合也有微小差异
```

**设计理由：** 单正弦曲线可被数学素养高的玩家在几回合内拟合出参数，导致"繁荣期放贷+衰退前收缩"成为支配策略。双周期叠加使波动模式不规则——有时两个周期共振产生深度衰退，有时互相抵消产生平坦期，拐点时机难以预测。

**涌现效果：**
- 双周期共振（两个都在下行）→ 罕见的深度衰退，大量公司同时陷入困境
- 双周期抵消（一升一降）→ 温和平稳期，竞争集中在银行间利率博弈
- 长周期拐点 → 短周期的"假复苏"或"假衰退"，误导依赖短期趋势判断的玩家

直接影响：标准利率、消费信心指数。
其他效果（公司投资意愿、原料价格等）通过经济传导自然涌现，不硬编码。

## 标准利率（派生）

模拟央行基准利率。衰退降息刺激经济，过热加息抑制，通胀时加息压制物价。

基于简化泰勒规则（Taylor Rule），同时考虑经济周期和通胀：

```
标准利率 = 基础利率 + α * 经济周期指数 + β * (通胀率 - 目标通胀率)

基础利率     = 中性利率（如 5%），经济中性且通胀达标时的利率
α（周期敏感度）= 经济周期波动对利率的影响幅度（如 3%）
β（通胀敏感度）= 通胀偏离目标时对利率的影响幅度（如 2%）
目标通胀率    = 健康通胀水平（如 2%）

例：
  繁荣期 + 通胀正常（周期+0.7, 通胀2%）:
    标准利率 = 5% + 0.7*3% + 0*(2%) = 7.1%（纯周期加息）

  周期中性 + 成本推动通胀（周期0, 通胀8%）:
    标准利率 = 5% + 0 + 2%*(8%-2%) = 17%（通胀驱动加息）

  衰退期 + 通缩（周期-0.7, 通胀-1%）:
    标准利率 = 5% - 0.7*3% + 2%*(-1%-2%) = 2.9% - 6% = -3.1% → clamp到下限（如 0.5%）
```

标准利率 clamp 在 [下限, 上限] 范围内，防止极端值（如深度衰退+通缩时利率为负）。

玩家的实际贷款利率 = 标准利率 + 玩家自定的风险溢价。

## 消费信心指数（派生）

居民采购阶段的驱动量，决定"居民愿意花多少钱"。

```
消费信心(t) = f(经济周期指数, 失业率)
```

经济周期上行 → 信心高 → 居民多花钱
经济周期下行 → 信心低 → 居民少花钱
失业率的影响留到居民系统一起设计。

## 产能利用率（统计）

反映供给侧健康度。

```
产能利用率(t) = 所有公司实际开工产能 / 所有公司最大产能
```

## 通胀率（统计）

反映物价变动速度，玩家判断贷款价值的参考。高通胀意味着借出去的钱贬值。

```
商品价格指数(t) = 加权平均（本回合所有终端消费品的成交均价）
通胀率(t) = (商品价格指数(t) - 商品价格指数(t-1)) / 商品价格指数(t-1)
```

## 生产总值（统计）

本回合所有公司的产出总价值，滞后指标。不影响其他系统，给玩家提供宏观经济判断依据。

```
生产总值(t) = Σ 每个公司本回合产出商品数量 * 该商品成交均价
```
