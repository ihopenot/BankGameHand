# Folk

居民部门，三个收入阶层，每层是独立的经济实体。

## 人口结构

三个阶层，各自持有 [EconomyEntryComponent](EconomyEntryComponent.md)，独立参与经济活动。

| 层级 | 人口占比 | 收入特征 | 消费特征 |
|------|---------|---------|---------|
| 低收入 | 大（如 60%） | 工资低，主要靠保底收入兜底 | 以刚需为主，几乎不买弹性品 |
| 中收入 | 中（如 30%） | 经济好时收入可观，衰退时缩减明显 | 刚需 + 弹性品，经济敏感 |
| 高收入 | 小（如 10%） | 收入高且稳定，受经济波动影响小 | 全品类消费，奢侈品主力 |

## 经济实体组件

每层各自持有：
[EconomyEntryComponent](EconomyEntryComponent.md)

三层独立核算资产负债表、现金、储蓄，独立通过 Transaction 接口交易。

## 每层收入

```
某层总收入(t) = 该层人口 * 人均工资(t) + 该层保底收入

人均工资(t) = 该层基础工资 * (公司工资总额(t) / 基准工资总额)
```

三层都有保底收入，由政府发放（见 [Goverment](Goverment.md)）：

| 层级 | 保底收入含义 | 金额 | 保底占总收入比例 |
|------|------------|------|----------------|
| 低收入 | 社会保障/最低生活保障 | 中等（确保买得起刚需） | 高 |
| 中收入 | 基础公共服务收入 | 较低 | 低 |
| 高收入 | 资本收益/投资回报 | 较高（确保维持奢侈品消费） | 中 |

经济波动对各层的冲击不同：
- 低收入层：基础工资低，保底收入占比大 → 收入波动小
- 中收入层：工资占比大，保底少 → 收入随经济明显波动
- 高收入层：基础工资高 + 资本收益保底 → 衰退时仍可维持奢侈品消费

## 每层消费需求

每层对每种终端商品有独立的需求：

```
某层对商品X的需求量 = 该层人口 * 人均基础需求 * (1 + 经济周期指数 * 经济敏感度)
```

不同层的人均基础需求不同：

| | 食物（刚需） | 手机（弹性） | 奢侈品 |
|---|---|---|---|
| 低收入 | 有需求 | 低需求 | 无需求 |
| 中收入 | 有需求 | 有需求 | 低需求 |
| 高收入 | 有需求 | 有需求 | 有需求 |

即使经济衰退，高收入层对奢侈品仍有基础需求，奢侈品公司不会直接死掉。

## 购买力约束（按层独立）

```
某层消费预算(t) = 该层总收入(t) * 该层消费倾向 + 可动用储蓄
```

如果总花费 > 消费预算：
- 按经济敏感度从低到高优先满足（先刚需后弹性品）
- 预算用完即停

如果消费预算 > 总花费：
- 多余进入该层储蓄（体现在该层 EconomyEntryComponent 的现金资产中）

## 采购分配

居民采购与公司采购共用同一套逐轮匹配机制（详见 [MainLoop](MainLoop.md) 采购阶段），但三层居民的偏好权重不同：

```
商品竞争力 = w_性价比 * (品质产出系数 / 标价) + w_品牌 * 品牌值
```

| 层级 | 价格敏感度 | 品牌权重 | 采购行为特征 |
|------|-----------|---------|-------------|
| 低收入 | 极高 | 极低 | 几乎纯看价格，选最便宜的卖方 |
| 中收入 | 中等 | 中等 | 性价比和品牌兼顾 |
| 高收入 | 较低 | 较高 | 偏好品牌和品质，对价格不敏感 |

每层居民作为一个整体参与采购匹配，按自己的偏好排序向卖方下单，通过逐轮匹配分配。

涌现行为：
- 经济繁荣时，高收入层拉动品牌厂商的需求和利润
- 经济衰退时，低收入层的刚需消费支撑便宜厂商存活
- 品牌厂商可以同时赚取高收入层的品牌溢价和中收入层的部分需求

## 与其他系统的衔接

- 居民 <- 公司：公司发工资（通过 Transaction 接口，公司付现金给对应层级的居民实体）
- 居民 -> 公司：居民消费（通过 Transaction 接口，各层居民实体独立付现金买商品）
- 居民 -> 经济指数：终端消费品成交均价 -> 通胀率；消费总额 -> 生产总值
- 居民 <-> 银行：储蓄存入银行，消费/信心不足时取款（见 [Bank](Bank.md) 居民取款机制）

## 通缩螺旋的自然刹车

公司发不出钱 -> 需求萎缩 -> 公司更差，这个循环由以下机制自然打破：
1. 保底收入保证最低消费不归零
2. 经济周期外生回升带动需求恢复
3. 衰退期利率下降，贷款成本低，银行介入的激励增大
4. 弱公司破产后供给减少，存活公司处境改善
