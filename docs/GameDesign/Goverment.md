# Goverment

政府，纯背景机制。不是经济实体，不持有 EconomyEntryComponent。资金无限，作为规则执行者。

## 职能一：保底收入发放

每回合向三个居民阶层发放保底收入：

| 层级 | 保底收入含义 | 金额 |
|------|------------|------|
| 低收入 | 社会保障/最低生活保障 | 中等（确保买得起刚需） |
| 中收入 | 基础公共服务收入 | 较低 |
| 高收入 | 资本收益/投资回报 | 较高（确保维持奢侈品消费） |

通过 Transaction 接口发放，政府侧资金无限，不校验余额。

## 职能二：产业政策

固定规则，游戏开始时配置，局内不变。

### 价格管制

```
某商品的价格上限 = 配置值（如食物最高价 50）
某商品的价格下限 = 配置值（如最低工资对应的最低售价）

卖方定价阶段，价格被钳制在 [下限, 上限] 范围内
```

### 产业补贴

```
某类商品的生产补贴 = 每单位产出补贴金额

每回合生产阶段结束后，政府向符合条件的公司发放：
补贴金额 = 该公司本回合产出量 * 单位补贴
```

## 执行时机

在主循环中的位置：
- 回合开始时：发放保底收入
- 公司定价时：价格管制生效（钳制价格）
- 生产阶段后：发放产业补贴
