# BankGameHand - 银行经营模拟游戏

一款回合制经济模拟策略游戏。玩家扮演银行家，在一个包含三条产业链的多行业经济体中经营银行，通过设定存贷利率、审批贷款等操作影响经济走势并实现盈利。

## 游戏概览

- **三条产业链**，涵盖从原材料到终端消费品的完整供应链：
  - **电子产业**：硅矿场 → 芯片工厂 → 手机工厂
  - **纺织产业**：棉花农场 → 纺织厂 → 服装厂
  - **食品产业**：麦田 → 面粉厂 → 食品厂
- **19 家初始企业**分布在各产业链环节，拥有独立的 AI 决策
- **3 个居民群体**（低/中/高收入），消费偏好各异
- **双周期经济模型**（短周期 25 回合 + 长周期 60 回合）模拟宏观经济波动
- 默认每局 **20 回合**（可配置）

## 回合流程

每回合按以下顺序执行：

1. **更新阶段** - 推进回合计数，更新经济指数，推进工厂建设
2. **出售阶段** - 企业根据库存和定价创建卖单
3. **购买阶段** - 居民和企业创建买单（CEO 特质影响采购偏好），市场撮合，交易结算
4. **生产阶段** - 工厂消耗原材料，产出成品
5. **决策阶段** - 企业 AI 调整定价，生成投资计划表（扩产/品牌/科技）
6. **玩家行动** - 展示经济摘要和企业数据，等待玩家操作
7. **结算阶段** - 生成账单并执行财务结算
8. **执行阶段** - 企业按投资预算执行扩产、品牌和科技投入

## 企业 AI 决策

每家企业由 5 维 CEO 特质驱动决策行为：

| 特质 | 影响 |
|------|------|
| 商业洞察力 | 定价精准度、市场前景感知偏差 |
| 风险偏好 | 涨价幅度、投资意愿、保留金比例 |
| 利润追求 | 降价保守程度 |
| 营销意识 | 品牌投入比例、采购时的品牌权重 |
| 科技重视度 | 科技投入比例 |

**决策流程**：plan_phase 生成投资计划表（不扣钱）→ act_phase 计算保留金 → 确定投资预算 → 全额执行或按比例分配 → 扩产不够建厂则回流。
